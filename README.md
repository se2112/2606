#  Детекция транспорта перед шлагбаумом

###  Описание
Программа в реальном времени анализирует видео с КПП и определяет наличие транспортного средства (автомобиль, грузовик, мотоцикл, автобус) **в зоне шлагбаума**, с **разделением по левому и правому въезду**.

Работа построена на модели **YOLOv8** с использованием **трекинга**, ROI-масок и логированием.

---

###  Основные возможности:
- Детекция и трекинг объектов в режиме реального времени.
- Разделение по двум зонам (левый и правый шлагбаумы).
- Определение типа ТС: car / truck / motorcycle / bus.
- Сохранение логов в CSV: время, класс ТС, зона.
- Гибкая настройка зон (включая поддержку многоугольников).
- Обработка перекрытий и ложных срабатываний.
